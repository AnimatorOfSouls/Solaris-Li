This page contains detailed explainations of all of the available commands.
If there are any commands missing, please contact AnimatorOfSouls#2980 on discord.
All commands use the prefix "cat", e.g. "cat help".


Game Information

help - View a list of commands.
newprofile - Creates a new profile if the user doesn't already have one.
profile - Use "p" for short. Shows the user's profile.
tiers - Shows the different tiers of cats which you can collect in order from the least rare to the most rare. More will be added in the future.



Game Commands

mine - Go mining for a kitty. The tier of kitty mined is selected based on RNG and the weights of each kitty.
inventory - Use "inv" for short. View inventory, currently just shows the number of each type of cat you have.
recipes - View available crafting recipes in the format "[RecipeID] cost -> product"
craft - Craft the recipes. Use in the format "cat craft RecipeID", e.g. "cat craft blue" to craft a bluecat at the cost of 2 whitecats
trade - Allows you to give other users cats from your inventory. Use in the format "cat trade @user cattype number", e.g. "cat trade @AnimatorOfSouls#2980 bluecat 5" to give AnimatorOfSouls 5 bluecats.
